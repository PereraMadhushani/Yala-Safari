# Yala Safari Website

Simple, modern React + Vite website for a Yala Safari service in Sri Lanka.

## Setup Instructions

1. Clone the repository:
	```bash
	git clone <repo-url>
	```

2. Go to the project folder:
	```bash
	cd "yala safari"
	```

3. Install frontend dependencies:
	```bash
	cd client
	npm install
	```

4. Start the development server:
	```bash
	npm run dev
	```

5. Open the app in your browser:
	```
	http://localhost:5173
	```

## Notes

- This repository currently contains the frontend app only (`client/`).
- There is no backend folder in the current workspace.
- If you typed `npm run dve`, use `npm run dev` instead.
